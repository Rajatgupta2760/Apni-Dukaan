<h2>🛒 Apni Dukaan (Django E-commerce Website)</h2>

<b>Apni Dukaan is a fully functional e-commerce web application built using Django. It allows users to browse products, manage their cart, place orders, and handle user authentication.</b>

<h3>🚀 Features</h3>
🛍️ Browse products<br>
🔍 View product details<br>
🛒 Add to cart & manage cart<br>
📦 Place and manage orders<br>
👤 User authentication (Login/Register)<br>
🧑‍💼 Admin panel for product & order management<br>
🖼️ Product image handling (media support)

<h3>🛠️ Tech Stack</h3>
Backend: Python, Django<br>
Frontend: HTML, CSS, JavaScript<br>
Database: SQLite<br>
Media Handling: Django Media Files<br>

<h3>⚙️ Installation & Setup</h3>
<h3>1. Clone the repository</h3> <pre><code>git clone https://github.com/your-username/apni-dukaan.git</code></pre> <h3>2. Navigate to the project</h3> <pre><code>cd apni-dukaan</code></pre> <h3>3. Create virtual environment</h3> <pre><code>python -m venv venv</code></pre> <h3>4. Activate environment</h3> <p><b>Windows:</b></p> <pre><code>venv\Scripts\activate</code></pre> <p><b>Mac/Linux:</b></p> <pre><code>source venv/bin/activate</code></pre>
<h3>5. Install dependencies</h3> <pre><code>pip install -r requirements.txt</code></pre> <h3>6. Apply migrations</h3> <pre><code>python manage.py migrate</code></pre> <h3>7. Run server</h3> <pre><code>python manage.py runserver</code></pre> <h3>8. Open in browser</h3> <pre><code>http://127.0.0.1:8000/</code></pre>
<hr>
🔐 Admin Panel

Access admin panel at:

http://127.0.0.1:8000/admin/

Create admin user:

python manage.py createsuperuser
